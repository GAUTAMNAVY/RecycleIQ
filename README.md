# ♻️ RecycleIQ  

**RecycleIQ** is an intelligent waste classification and awareness system built to promote sustainable waste management practices. It combines machine learning, AI, and modern web technologies to educate users, classify waste, and guide them toward eco-friendly choices.
<img width="1832" height="756" alt="image" src="https://github.com/user-attachments/assets/afc64a2f-e41f-46b8-bbb8-01e5b4f68c1c" />
## 🌟 Features  

## 1. 🧠 Text-Based Waste Classification
Classifies waste items based on user input using a hybrid ML model combining Random Forest and Logistic Regression. Helps users identify the correct waste category for disposal.
<img width="1353" height="690" alt="image" src="https://github.com/user-attachments/assets/3a40f909-f6ef-4f45-aee2-01f1302c58fc" />

## 💼 Tech Stack  

| Category           | Technologies Used |
|--------------------|-------------------|
| **Frontend**       | Next.js, Tailwind CSS |
| **Backend**        | Node.js, Express.js, Python (for ML integration) |
| **Database**       | PostgreSQL |
| **Machine Learning** | Scikit-learn (Random Forest + Logistic Regression) |
| **AI/LLMs**        | Google Gemini API, ChatGPT API, LangChain |
| **Content Moderation** | Google Gemini API |
| **Mapping & Location** | Geolocation API, Leaflet.js / Mapbox |

## 2. ♻️ Eco-Friendly Alternatives
Suggests alternative eco-friendly products using Google Gemini for items that are harmful to the environment.
<img width="1139" height="396" alt="image" src="https://github.com/user-attachments/assets/ea180660-2384-4a8e-8b40-01e2a419d2bf" />

## 3. 📝 Full Blog System
Users can add, delete, like, dislike, and comment on posts.
Comments can also be deleted by the user who posted them.
Platform to share recycling tips, sustainability practices, and local initiatives
<img width="1365" height="698" alt="image" src="https://github.com/user-attachments/assets/c41cbf01-172f-45e9-a154-0b27d2815235" />
## 4. 🚫 Blog Content Moderation
All blog content is filtered through Google Gemini to ensure that posts are related to recycling, waste management, or sustainability—keeping the platform focused and clean.
## 5. 🤖 Interactive Chatbot
An AI-powered assistant that can answer basic recycling-related questions in short responses using a combination of ChatGPT API, Gemini API, and LangChain. It engages users with recycling facts and quizzes.
<img width="355" height="609" alt="image" src="https://github.com/user-attachments/assets/14f1a6d5-54dd-45ae-98df-7cfe8f46144c" />
## 6. 🧾 "Chat with AI" Feature
A dedicated chat experience where users can ask detailed queries and get brief but informative responses using Gemini.
<img width="1920" height="922" alt="image" src="https://github.com/user-attachments/assets/579df973-e093-4a06-8dfc-473cc63eda38" />
## 7. 📰 Waste Management News Feed
Shows real-time news on sustainability and waste management fetched from trusted external APIs. Users can subscribe to the newsletter and will receive 1 news article on their email daily. They can unsubscribe from email itself and from user portal also.
<img width="963" height="512" alt="image" src="https://github.com/user-attachments/assets/8e1db4a8-ebf7-47e8-94f0-77ec3b3eb88b" />
## 8.📍 Nearby Recycling Center Locator
Finds recycling centers near your location using geolocation APIs and displays them on an interactive map.

Admins can add, update, and manage data of recycling centers to keep the database accurate and up-to-date.
